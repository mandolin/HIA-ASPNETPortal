
0.You need NuGet enabled.

1. Run the scripts in the Setup directory: Portal_CreateDB.sql, Portal_LoadConfig.sql, Portal_LoadData.sql
2. Copy src/Portal/Config/Templates/connectionStrings.config to {UserProfile}/Web/HIA-ASPNETPortal/dev/connectionStrings.config and update the Portal connection string
3. Build and Run. Login as admin with admin/admin

PLEASE note that this version is under construction. For a stable version download the release.
