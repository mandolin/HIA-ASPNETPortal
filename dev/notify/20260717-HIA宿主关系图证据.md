# HIA 宿主关系图证据通知

## 背景

HIA Documentation Sys 在 `W-P22.5` 中使用 HIA-ASPNETPortal 的 WorkZone 临时文档产物完成了第一轮宿主关系图验证。

本通知只记录 HIA 侧验证结论，没有修改 HIA-ASPNETPortal 的源码、配置、依赖、CI 或生成产物。

## 已验证内容

- 统一文档输出包含 1547 个 project entries。
- `project-index.json.relationGraph` 包含 1650 个 nodes、1595 条 relations。
- relation kind：
  - `documents-endpoint` 48 条。
  - `documents-source` 48 条。
  - `semantic-member` 1499 条。
- LSP host 可读取 workspace/opened `project-index.json` relation graph。
- VS Code relation helper 可为 1595 条 relation 生成 choices 与 navigation targets。
- Browser panel 可通过 `--project-index` 消费同一 relation graph，并输出 relation list payload。

## HIA 侧证据位置

- `HIA-Documentation-Sys/work-zone/temp/aspnetportal-unified-html/current`
- `HIA-Documentation-Sys/work-zone/temp/target-host-relation-evidence/current/target-host-relation-evidence.json`

## 建议动作

- 暂无必须立即执行的代码变更。
- 后续如需在 HIA-ASPNETPortal 内正式接入，可由项目侧评估是否复制或吸收 HIA WorkZone 中生成的候选配置、脚本和依赖版本。
- 若后续升级 `@hia-doc/*` 依赖、启用 CI gate 或调整 XML documentation 输出路径，建议单独开分支处理，避免与当前业务或文档化改动混合。
